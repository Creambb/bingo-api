# bingo-api
一个基于小程序的文具商城--后台api

### 依赖安装

```
    // 初始化项目
    npm init -y
    // 安装express
    npm install express --save
    // 安装mysql
    npm install --save mysql
    // 安装sequelize
    npm install sequelize --save
    // sequelize需要单独安装mysql2依赖，否则报错'Please install mysql2 package manually'
    npm install mysql2 --save
```
